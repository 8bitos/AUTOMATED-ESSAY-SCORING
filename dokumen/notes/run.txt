Cara Menjalankan Project SAGE-Skripsi

1) Jalankan Docker daemon (kalau belum aktif)
   systemctl start docker

2) Dari root project, jalankan container
   cd "/run/media/sapta/Data/SAGE (Copy)/SAGE-Skripsi"
   docker compose up -d

3) Cek status container
   docker compose ps

Service yang naik dari docker-compose:
- PostgreSQL: localhost:5433
- Redis: localhost:6380
- Frontend (Next.js): http://localhost:3001

4) Jalankan backend Go (manual, tidak ada di docker-compose ini)
   cd backend
   go mod tidy
   go run .

Backend API jalan di:
- http://localhost:8080
- Base API frontend: http://localhost:8080/api

5) Cara stop (lengkap)
   a. Stop backend Go (yang jalan manual):
      - Di terminal backend, tekan Ctrl + C
      - Atau pakai:
        pkill -f "go run ."

   b. Stop 1 service tertentu saja (tanpa hapus container):
      cd "/run/media/sapta/Data/SAGE (Copy)/SAGE-Skripsi"
      docker compose stop db
      docker compose stop redis
      docker compose stop web-frontend

   c. Start lagi service yang di-stop:
      docker compose start db
      docker compose start redis
      docker compose start web-frontend

   d. Stop semua service compose (container tetap ada):
      docker compose stop

   e. Stop + hapus semua container project:
      docker compose down

   f. Stop + hapus semua container + volume data:
      docker compose down -v

docker stop $(docker ps -q)

Catatan env penting:
- backend/.env -> DB_PORT=5433, FRONTEND_ORIGIN=http://localhost:3001
- frontend env (jika run lokal non-docker) -> NEXT_PUBLIC_API_BASE_URL=http://localhost:8080/api
